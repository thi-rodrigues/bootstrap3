# BootStrap3
    Framework para desenvolvimento de componentes de interface e frontend para sites e aplicações web. Usa `HTML`, `CSS` e `JavaScript`.

- Instalação:
    npm install bootstrap

## Grid
    col-xs: <= 480px
    col-sm: >= 768px
    col-md: >= 998px
    col-lg: >= 1200px
